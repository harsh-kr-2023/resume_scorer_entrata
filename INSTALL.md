# Installation Steps

## Step 1
Extract the zip file

## Step 2
Open folder in Command Prompt

## Step 3
```cmd
run.bat
```

## Step 4
Edit `.env` file and add your OpenAI API key:
```
LLM_API_KEY=sk-your-actual-openai-key
```

## Step 5
```cmd
start.bat
```

## Step 6
Open `frontend/index.html` in browser
